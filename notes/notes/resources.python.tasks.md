## Practical Tasks

1. **Variable Practice** – Write a calculator that uses all basic data types.
2. **Control Flow** – Build a FizzBuzz and a prime number checker.
3. **Functions** – Write a function that flattens nested lists with recursion.
4. **Modules** – Create a custom module with math helpers and reuse it.
5. **OOP** – Build a `BankAccount` class with `deposit()`, `withdraw()` methods.
6. **Error Handling** – Write a robust CSV parser that handles malformed rows.
7. **File I/O** – Script to scan a directory and summarize file types and sizes.
8. **venv/pip** – Create a virtual environment and install `requests`, `pandas`, etc.

> Bonus: Automate a daily task you perform manually using Python.
